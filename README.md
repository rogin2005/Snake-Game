# Snake-Game

Jogo da cobrinha feito com html, css e js

## Instalação
1. Clone o repositório:
```
bash

git clone https://github.com/rogin2005/Snake-Game.git
```
2. Para jogar basta abrir o arquivo "index.html" no seu navegador ou ainda utilizar a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) do Vscode

3. O jogo funciona com as teclas w(cima), a(esquerda), s(baixo) e d(direita), mas pode ser alterado para as setas direcionais do teclado, nas linhas finais do script.js tem comentários sobre as direções do jogo nos trechos onde tem " if (key == "d" " em cada um podemos informar o valor das setas que são: "ArrowDown"(baixo), "ArrowUp"(cima), "ArrowRight"(direita) e "ArrowLeft"(esquerda). em cada direção podemos alterar o valor que ele compara com key conforme a direção e assim fazer a troca das teclas, claro que pode ser adicionado qualquer outra tecla. [Documentação eventos de teclado](https://developer.mozilla.org/en-US/docs/Web/API/UI_Events/Keyboard_event_key_values)
